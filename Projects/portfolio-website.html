Portfolio Website

Category: Web Design & Development

Project Overview

This project is a clean, mobile-first personal portfolio site that showcases my web development and graphic skills. It includes sections like About Me, Projects, Education, and Contact.

Technologies Used

  • HTML5, CSS3, JavaScript
  • Bootstrap 5 for responsiveness
  • Scroll animations and smooth navigation

Planning & Execution

I began with wireframes in Figma, then built a layout with Bootstrap. I focused on clean design, fast loading, and SEO-friendly markup.

Features

  • Responsive across all devices
  • Downloadable resume (PDF)
  • Project detail pages with deep insights
  • Modern color palette and typography
